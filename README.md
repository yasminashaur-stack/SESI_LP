# Git bash / Comandos e explicações
**git config --global user.name "seu nome"** define o nome do usuário no Git
**git config --list** mostra todas as configurações do Git (nome, e-mail e outras configurações salvas)
**git config --global user.email"nome do email"** define o e-mail do usuário

# Git rub / comandos e explicações
**Git clone "link"** baixa (copia) um projeto do GitHub
**cd SESI_LP** cria a pasta
**git status** mostra o estado dos arquivos
**git add nome do arquivo ou (git add asterisco)** adiciona arquivos para preparar o envio (commit)
**git commit -m "descrição"** salva as alterações com uma descrição
**git push** envia as alterações para o repositório no GitHub
**git pull** atualiza o projeto, trazendo as mudanças do repositório no GitHub para o seu computador
